# 🤝 Contributing

Thank you for considering contributing! Here's how to help:

## 🐛 Reporting Bugs
1. Check existing [Issues](../../issues) first
2. Open a new issue with the bug template
3. Include: OS, Python version, steps to reproduce

## 💡 Suggesting Features
1. Check existing [Issues](../../issues) and [Discussions](../../discussions)
2. Open a feature request with clear use case

## 🔄 Pull Requests
1. Fork the repository
2. Create a feature branch
3. Make changes with clear commits
4. Update documentation if needed
5. Submit PR with description

## 🧪 Testing
Before submitting:
- Test with domain + IP targets
- Verify JSON output is valid
- Ensure no new syntax errors

## 📐 Code Style
- Follow PEP 8 (120-char lines)
- Use type hints for public functions
- Meaningful variable names

Questions? Open a [Discussion](../../discussions). 🚀
